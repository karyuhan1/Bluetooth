메인 엑티비티 - 시작 화면으로 블루투스 권한 확인 및 권한 요쳥을 담당합니다. / 시작 버튼을 누르면 스캔 액티비티로 이동합니다ㅏ
스캔 액티비티 - 어플리케이션에 설정된 UUID 와 맞는 블루투스 기기를 검색하여 가져오고 해당 리스트 클릭시 기기와 연결을 시도합니다.
워크 액티비티 - 시작 버튼을 누르면 연결된 블루투스 기기의 센서값을 읽어와 저장하며 지정된(현재는 6천개)데이터량을 수집하면 자동으로 러닝액티비티로  이동합니다. /작동중 리셋 버튼을 누르면 초기화 합니다.
러닝 액티비티 -  db에 저장된 데이터를 기반으로 텐서플로우 모델을 활용하여 학습하며 학습결과를 db에 저장합니다.
액추얼 액티비티 - db에 저장된 모델을 불러와 착용자의 센서를 조작하여 평의를 돕습니다.
데이터베이스 헬퍼  - db관련 명령을 집합해 놓은 헬퍼 클래스입니다. 기본적으로 에셋 폴더에 있는 db를 가져와서 사용합니다.
네비게이터 헬퍼 - 액티비티 이동 메서드를 저장해놓았습니다.
퍼미션 헬퍼 - 블루투스 관련 권한을 확인하는 헬퍼 클래스 입니다.


궁금한점 - 코드를 지금 챗지피티와 구글링으로 코드를 작성했는데 전체적으로 코드리뷰를 한번 부탁드리고 싶습니다.
피그마를 이용해서 안드로이드 스튜디오 xml 레이아웃을 바로 적용시키는 방법을 알고 싶습니다.
자체적으로 인식한 문제
- 스캔 액티비티에서 학습된 모델이 있을시 바로 액추얼 액티비티로 이동
- 액추얼 액티비티에서 학습 다시하기 기능을 추가할 예정
